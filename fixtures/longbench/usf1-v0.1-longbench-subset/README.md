# Unicode Snapcompact F1 LongBench Subset v0.1

Source: LongBench (<https://huggingface.co/datasets/zai-org/LongBench>).

This subset contains 35 deterministic cases: 5 from each selected task. Each case stores LongBench input and context in `context.txt`, one QA item in `recall_qa.json`, and source metadata beside it.

Exact tasks: passage_retrieval_zh, lcc, repobench-p. Other tasks are semantic answer-span tasks.
